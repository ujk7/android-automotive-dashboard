# Android Automotive Dashboard

A production-style Android Automotive dashboard built with modern Android technologies.

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM
- StateFlow
- Hilt
- Repository Pattern
- Android Automotive Simulation

## Features

- Digital Speedometer
- Analog Speedometer
- Fuel Monitoring
- Battery Monitoring
- Temperature Monitoring
- Gear Indicator
- Door Status

## Architecture

UI
↓
ViewModel
↓
Repository
↓
DataSource
↓
AutomotiveManager
